---
title: Textarea
---

A standard textarea. Use for multi-line text. Manipulate the height with the `rows` attribute or the following modifier classes: `.c-input--h-small`, `.c-input--h-medium`, `.c-input--h-large`.

Optionally add a class of `.autosize` to autosize. This functionality relies on <a href="https://www.npmjs.com/package/autosize">autosize</a>.
